# 📱 First Gameplay Screen

## Scene
The Bridge of Ember Creek

---

## Goals
- Calm and readable UI
- Support voice and drawing input
- Large interaction areas for kids
- Minimal distractions

---

## Screen Layout

### 1. Narration Area
Displays story text and encounter narration.

Example:
"The dragon roars!"

---

### 2. Scene Artwork
Displays:
- bridge
- dragon
- fire

Static artwork for MVP.

---

### 3. Prompt Area
Displays player instruction.

Examples:
- "Say FIRE"
- "Draw a shield"
- "DEFEND!"

---

### 4. Input Area
Canvas area for drawing symbols or gestures.

Examples:
- line
- triangle
- wave

---

### 5. Action Buttons
- Voice
- Draw
- Help

---

## UI Design Principles

- Large readable text
- Minimal clutter
- Calm color palette
- Soft transitions
- Kid-friendly spacing

---

## MVP Notes

No advanced animation required.

Focus on:
- interaction
- readability
- core gameplay loop
